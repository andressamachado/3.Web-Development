What are main benefits of using CSS?

Cascading Style Sheets are use to improve the appearence of the web content once it gives more formatting control than HTML. CSS also has something called stylesheet that allows the web designer to focus all the formatting in that file. It helps at the time of review or even modify, once the maintenance would be in just one file.Because all the presentation is developed in a separate file, the HTML files that are driven by the CSS StyleSheet, work better with accessibility tools, such as screen reader. Enriching experience.That is the same reason that makes those pages faster to download.Because of this feature, CSS may be used to adapt a page to respond to different browsers and window sizes.

What are different parts of a CSS style rule?

Selector and a series of property. A property is a pair of values that together are called as a declaration and a selector is what identifies the elements that will be affected in the HTML.

What are an element selector and grouped element selector in CSS? Provide an example of each.

An elemente selector is the begining of the CSS rule. As said before, the selector is what identifies what will be affected in the HTML file by the declarations. An exemple of selector is the body tag name. To select all the elements, an * can be used as a selector. On the other hand, a grouped element selector is when a group of selectors shared the same declarations. For example:
header, nav, main { padding: 20px 30px; }

What are class selectors in CSS? What are id selectors in CSS?

Class selectors are used to affect class elements in the HTML file. To select elements with a specifi class is necessary use a dot + the name of the class. Class can be used in a multiple HTML elements. For example:
.myclass { background: white; }
An ID Selector is used to affect an element that was labeled with an id attribute. It is possible to have just one id of given type.

What are contextual selectors in CSS?

It is a feature that allows to select elements based on their ancestors, descendants, or siblings.

What are the three cascade principles used by browsers when style rules conflict?

Inheritance, Specificity, and Location.
